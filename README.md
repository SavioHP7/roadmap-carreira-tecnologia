# Roadmap de Carreira em Tecnologia

Este repositório contém **prompts estruturados** para auxiliar na descoberta e planejamento de carreiras em tecnologia.  
Ele foi dividido em duas etapas principais:

---

## 📌 Estrutura

### 1. Entrevistador de Carreira (Agent 1)
- Conduz uma entrevista de 7 perguntas para entender perfil, interesses e objetivos.
- Analisa as respostas e sugere 3 carreiras ranqueadas.
- Transfere para o planejador especializado (Agent 2).

### 2. Planejador de Carreira (Agent 2)
- Recebe as informações do entrevistador.
- Gera um **plano completo de estudos** com:
  - Visão do dia a dia
  - Mapa de skills
  - Roadmap de 90 dias
  - Projeto de portfólio
  - Roteiro de entrevistas
  - Trilha recomendada na DIO

---

## 🎯 Objetivo

O objetivo deste repositório é servir como **guia prático** para quem deseja:
- Descobrir a carreira ideal em tecnologia.
- Estruturar um plano de estudos personalizado.
- Criar projetos de portfólio e se preparar para entrevistas.

---

## 🚀 Como Usar

1. Leia o prompt do **Entrevistador de Carreira** e responda às 7 perguntas.
2. Analise as carreiras sugeridas e escolha uma.
3. Use o prompt do **Planejador de Carreira** para gerar seu roadmap completo.
4. Siga o plano e registre sua evolução.

---

## 📂 Conteúdo

- `agent1_entrevistador.md` → Prompt para entrevista e análise de perfil.
- `agent2_planejador.md` → Prompt para criação de roadmap personalizado.
